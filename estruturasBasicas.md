estrutura inicial:
```bash
<!DOCTYPE html>
<html> "código.aqui" </html>
```
cabeça e corpo do código:
```bash
<head> 
<title> "título.do.código" </title> 
</head>
```
títulos e subtitulos:
```bash
<h1> </h1>
<h2> </h2>
<h3> </h3>
<h4> </h4>
<h5> </h5>
<h6> </h6>
```
parágrafos:
```bash
<p> </p>
```
quebra de linha:
```bash
<br> </br>
```
palavra em negrito:
```bash
<strong> </strong>
```
palavra em italico:
```bash <em> </em>
```
pedaço de estrutura usado para separar estrutas de um código e facilitar estilizacão:
```bash
<div id="definir.id"> </div>
```
